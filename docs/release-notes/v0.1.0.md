# ReaPack Porter v0.1.0

The first standalone cross-platform release of ReaPack Porter.

## Highlights

- Native standalone GUI for Linux, Windows and macOS
- Separate frozen CLI
- Safe repository export and import
- Folder and ZIP bundles
- Preview and dry-run support
- REAPER process guard
- Automatic path detection
- Timestamped backups and atomic writes
- SHA256 checksums for every platform archive

## Downloads

- Linux x86_64: `ReaPack-Porter-0.1.0-linux-x86_64.tar.gz`
- Windows x86_64: `ReaPack-Porter-0.1.0-windows-x86_64.zip`
- macOS Intel x86_64: `ReaPack-Porter-0.1.0-macos-x86_64.zip`
- macOS Apple Silicon arm64: `ReaPack-Porter-0.1.0-macos-arm64.zip`

## Important

- Close REAPER completely before importing.
- Export is safe while REAPER is running.
- Windows and macOS builds are unsigned.
- macOS builds are not notarized.
- Verify the SHA256 checksum after downloading.

## After Import

Start REAPER and run:

```text
Extensions > ReaPack > Synchronize packages
```

## Legacy Lua Version

The original `reapack_porter.lua` remains available in the repository.
